# MAXIHOME Ecommerce

## Descripción del proyecto

MAXIHOME Ecommerce es un proyecto de comercio electrónico que ofrece funcionalidades básicas de administración de productos, catálogo de productos y autenticación de usuarios. El objetivo principal es implementar un CRUD completo y mostrar los productos en un catálogo, permitiendo que los usuarios administradores realicen acciones de administración mientras que los usuarios visitantes solo puedan ver las publicaciones. Además, hay requerimientos opcionales como el registro de usuarios y la funcionalidad de lista de productos deseados.

### Estructura general del sitio

El diseño del sitio es completamente responsive. Se desarrollaron las siguientes páginas:

- **Página Principal:** Muestra un catálogo de productos cargados y permite realizar búsquedas por nombre o categoría.
- **Página de administración:** Permite a los usuarios administradores agregar, borrar y editar productos y usuarios registrados. Muestra una tabla con los productos cargados y usuarios registrados.
- **Página de detalle:** Muestra información detallada de un producto seleccionado, incluyendo nombre, precio, imagen, descripción y cantidad en stock.
- **Página acerca de nosotros:** Proporciona información sobre el equipo que desarrolló el sitio, incluyendo una tarjeta con la foto de cada miembro del equipo.
- **Página "en construcción":** Es una página personalizada para enlaces o botones que no tienen una funcionalidad establecida por el momento.
- **Login:** Página para que los usuarios administradores inicien sesión.

### Instrucciones de instalación

El proyecto es una página web y no requiere instalación. Para ejecutarlo, simplemente abre el archivo HTML principal en un navegador web compatible, o se ingresa en la siguiente URL https://mxhome.netlify.app/
